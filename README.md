각 서브디렉토리는 자체 `CLAUDE.md`(AI 작업 규칙)와 `README.md`(프로젝트 개요)를 보유합니다.

저장소 전체에 적용되는 컨벤션(브랜치 전략, CI/CD 정책, 디렉토리 구조 규칙 등)은
[`CLAUDE.md`](CLAUDE.md)를 참고하세요.
