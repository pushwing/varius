# 동선 애니메이션 재생 (Route Playback) 설계

- 날짜: 2026-07-23
- 대상: `Iter/` 메인 지도(`app/Views/map.php`)
- 상태: 설계 승인됨

## 목적

메인 지도에서 날짜를 선택하면 그 날의 이동 동선을 시간순 애니메이션으로 재생한다.
이동 마커가 경로를 따라 움직이고, 지나온 구간의 경로선이 점진적으로 그려져
"그 날 하루의 여정"을 시각적으로 되짚을 수 있게 한다.

## 요구사항 (확정)

| 항목 | 결정 |
|------|------|
| 재생 단위 | **하루 단위** — 사이드바에서 선택한 날짜의 동선만 재생 |
| 화면 연출 | **마커 이동 + 경로 점진 그려짐** + 사진 클러스터 지점 도달 시 마커 펄스 강조 (카메라 추적 없음) |
| 속도 매핑 | **거리 비례·고정 재생시간** — 하루 전체 15초(1x), 구간별 소요는 하버사인 거리에 비례 |
| 컨트롤 | **재생/일시정지 + 배속(1x/2x/4x)** — 진행바 탐색은 범위 제외 |

## 구현 접근 — 순수 JS + requestAnimationFrame (채택)

외부 플러그인 없이 Leaflet 기본 API만 사용한다.

- 진행 경로선: `L.polyline([]).setLatLngs(...)` 로 프레임마다 연장
- 이동 마커: `marker.setLatLng(...)` 로 프레임마다 이동
- 타이밍: `requestAnimationFrame` 루프에서 프레임 델타 누적

### 기각한 대안

- **Leaflet 플러그인 조합(MovingMarker + SnakeIn)**: 수년째 미유지보수, 마커·선 동기화가 억지스럽고 배속·거리 비례 타이밍 커스텀 불가, CDN 의존성 추가.
- **Canvas/deck.gl**: 이 요구사항엔 오버킬.

## 상세 설계

### 범위

프론트엔드 전용 — 백엔드 변경 없음. routes API(`GET /routes`)가 이미 날짜별
시간순 좌표(`dates[].points`)를 내려주므로 그대로 사용한다.

### UI

- 사이드바에서 날짜 선택(`selectDay`) 시 지도 하단에 재생 컨트롤 바 표시.
  - 구성: ▶(재생/일시정지 토글) + 배속 버튼(1x → 2x → 4x 순환).
  - 해당 날짜 좌표가 2개 미만이면 컨트롤을 표시하지 않는다.
- 재생 시작 시 해당 날짜의 정적 경로선 opacity 를 낮춰 진행선과 구분한다.
- 진행 경로선은 같은 날짜 색상, 정적 선보다 굵게(weight 5) 그린다.

### 재생 로직

1. 선택 날짜의 `points`를 구간(연속 좌표쌍) 목록으로 변환한다.
2. 각 구간의 하버사인 거리를 계산해 총 거리 대비 비율로 재생시간 15초(1x 기준)를 배분한다.
   - 거리 0m 구간(같은 지점 연속 사진)은 스킵한다.
3. rAF 루프: 프레임 델타 × 배속을 누적 → 현재 구간과 구간 내 진행률 산출 →
   선형 보간 좌표로 마커 이동 + 진행선 연장.
4. 구간 경계(사진 지점) 도달 시, 그 지점에서 30m 이내에 중심을 둔 같은 날짜의 클러스터 마커가 있으면
   짧은 펄스 강조를 준다(클러스터는 30m 반경으로 묶이므로 좌표가 정확히 일치하지 않을 수 있다).
   같은 클러스터에 연속으로 도달하면 펄스는 한 번만 준다.
5. 종료 시(끝 도달·다른 날짜 선택·정지): 진행 레이어(마커·진행선) 제거, 정적 경로선 opacity 원복.

### 컨트롤 동작

- **일시정지**: rAF 는 유지하되 델타 누적을 중단(또는 루프 중단 후 재개 시 타임스탬프 리셋).
- **배속**: 델타에 배수 적용 — 재생 중 변경 즉시 반영.
- **날짜 전환**: 재생 중 다른 날짜를 선택하면 현재 재생을 정리(cleanup)하고 새 날짜 기준으로 컨트롤을 재표시.

### 엣지 케이스

- 탭 비활성화: rAF 가 멈췄다 복귀하면 큰 델타가 들어오므로 프레임 델타를 100ms 로 클램프해 순간이동을 방지.
- 좌표 1개뿐인 날: 컨트롤 미표시(재생할 경로 없음).
- 전체 구간 거리 합이 0(모든 사진이 같은 지점): 컨트롤 미표시.

### 코드 구조

- `app/Views/map.php` 인라인 스크립트에 추가(기존 패턴 유지).
- 구간 시간 배분·선형 보간은 DOM/Leaflet 비의존 순수 함수로 분리해 가독성을 확보한다
  (`buildSegments(points, durationMs)`, `interpolate(seg, t)` 등).

## 검증

- 이 프로젝트는 JS 테스트 인프라가 없다 — 프로젝트 규칙대로 **실제 브라우저 구동으로 수동 확인**한다.
  - 재생/일시정지/배속 동작, 진행선·마커 동기화, 날짜 전환 시 정리, 펄스 강조.
- PHP 변경이 없으므로 `composer ci` 는 회귀 없음 확인용으로만 실행한다.
