# 개인정보처리방침

**시행일자**: 2026-07-20

본 개인정보처리방침(이하 "방침")은 Iter(이하 "서비스")가 이용자의 개인정보를 어떻게 수집·이용·보관·파기하는지 설명합니다.
서비스는 Google Photos Picker API를 통해 이용자가 직접 선택한 사진의 위치정보(GPS)를 추출하여
지도 위에 날짜별 이동 동선을 시각화하는 기능을 제공합니다.

## 1. 수집하는 개인정보 항목

| 구분 | 항목 | 수집 방법 |
|------|------|-----------|
| 인증 정보 | Google 계정 식별자, OAuth Access/Refresh Token | Google OAuth2 로그인 |
| 사진 메타데이터 | 촬영 위치(위도·경도), 촬영 시각, Google Photos 미디어 항목 ID | Google Photos Picker로 이용자가 직접 선택한 사진의 EXIF |
| 파생 이미지 | 선택한 사진의 가로 300px 썸네일 | 서버에서 원본 다운로드 후 생성 |
| 서비스 이용 기록 | 접속 로그, IP 주소, 요청 시각 | 서비스 이용 과정에서 자동 생성 |

서비스는 **Google Photos Picker에서 이용자가 명시적으로 선택한 사진에만 접근**하며,
이용자의 전체 사진 라이브러리에 임의로 접근하지 않습니다.
Google Photos API 응답 자체에는 GPS 좌표가 포함되지 않으므로, 위치정보는 이용자가 선택한
사진 원본 파일의 EXIF 데이터를 서버에서 직접 파싱하여 얻습니다.

## 2. 개인정보 수집·이용 목적

- Google 계정 인증 및 서비스 로그인 유지
- 선택된 사진의 GPS·촬영시각 추출 및 날짜별 이동 동선 지도 시각화
- 지도 미리보기용 썸네일 제공
- 서비스 부정 이용 방지(요청당 처리 매수 제한, 세션 생성 횟수 제한 등 레이트 리밋)

## 3. 개인정보의 보유 및 이용 기간

| 항목 | 보유 방식 | 보유 기간 |
|------|-----------|-----------|
| OAuth Refresh Token | AES 등으로 **암호화**하여 DB 저장 | 이용자가 연동 해제 또는 계정 삭제 요청 시까지 |
| GPS 좌표·촬영시각 | DB(`photo_locations`) 저장 | 이용자가 삭제 요청 시까지 |
| 사진 원본 파일 | **저장하지 않음** — EXIF 추출 직후 즉시 폐기 | 처리 완료 즉시 삭제 |
| 썸네일(300px) | 서버 파일시스템 저장 | 원본 데이터 삭제 요청 시 함께 삭제 |

- **사진 원본 이미지는 서버에 저장되지 않습니다.** 위치정보 추출을 위해 일시적으로 다운로드한 원본은
  EXIF 파싱과 썸네일 생성이 끝나는 즉시 삭제됩니다.
- 지도 미리보기를 위한 가로 300px 썸네일만 예외적으로 보관되며, 풀사이즈 원본이 다시 필요한 경우
  Google Photos API를 통해 그때그때 재조회합니다.

## 4. 개인정보의 제3자 제공

서비스는 이용자의 개인정보를 **원칙적으로 제3자에게 제공하지 않습니다.**
다만 다음의 경우는 예외로 합니다.

- 이용자가 사전에 별도 동의한 경우
- 법령에 근거하거나 수사기관이 법령에 정해진 절차에 따라 요청한 경우

## 5. 개인정보 처리 위탁

서비스는 원활한 운영을 위해 아래와 같이 개인정보 처리 업무를 외부에 위탁할 수 있습니다.

| 수탁자 | 위탁 업무 내용 |
|--------|----------------|
| Google LLC | OAuth2 인증, Google Photos Picker API를 통한 사진 접근 |
| [클라우드/호스팅 제공자] | 서버 인프라 운영, 데이터베이스 호스팅 |

## 6. 이용자의 권리와 행사 방법

이용자는 언제든지 아래 권리를 행사할 수 있습니다.

- 개인정보 열람·정정·삭제 요청
- Google 계정 연동 해제 및 서비스 탈퇴
- 저장된 GPS·동선 데이터 전체 삭제 요청

권리 행사는 `advisor@aivance.kr` 로 요청하실 수 있으며, 서비스는 관련 법령이 정한 기간 내에 조치합니다.
또한 Google 계정 자체의 앱 접근 권한은 [Google 계정 권한 관리 페이지](https://myaccount.google.com/permissions)에서
직접 철회할 수 있습니다.

## 7. 개인정보의 안전성 확보 조치

- OAuth Refresh Token 등 민감 인증정보는 암호화하여 저장합니다.
- 시크릿(Client ID/Secret, 암호화 키 등)은 환경변수로만 관리하며 코드에 하드코딩하지 않습니다.
- 사진 원본은 처리 후 즉시 파기하여 불필요한 보관을 최소화합니다.
- 응답 및 로그에 인증 토큰 등 민감정보를 노출하지 않습니다.
- API 요청에는 레이트 리밋을 적용하여 비정상적인 대량 접근을 방지합니다.
